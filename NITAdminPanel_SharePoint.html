Admin Access
Who can sign in to this panel

Access to this IT Admin Panel is controlled by Microsoft 365 sign-in. Only accounts whose email address is listed in the GU_Admins SharePoint list (column: Email) can log in.

To add or remove an admin, edit the GU_Admins list directly in SharePoint — add or remove rows with the admin's Microsoft 365 email address in the Email column.

🏦 Branch Operations
Reset Activation Key
Issue a new key for re-activation
Reset Branch Login
Force manager to set new password

Clears password hash and sets FirstLogin=TRUE so the manager must set a new password on next login.

Transfer Manager
Move manager to another branch

Old branch becomes vacant. History saved as TRANSFERRED.

⚠️ Critical Actions — Use with Caution
Manager Changed / Deactivate
Use when a manager resigns or is replaced

This revokes the saved Chrome signature, clears login credentials, and forces full re-activation for the branch workstation. This action cannot be undone — the branch will need to re-activate from scratch.
